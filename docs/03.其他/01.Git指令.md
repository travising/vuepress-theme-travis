## 第一次提交
```
git init
git add -A
git commit -m **

# 添加远程仓库
git remote add origin https://github.com/gcfae/dms.git

# 查看远程仓库
git remote -v

# 提交到本地主分支(master)到远程仓库（origin）
git push -u origin master
```

## 常规提交
```
git init
git add -A
git commit -m **
git push origin master
```
