## :star: Markdown扩展语法

### :package: 自定义容器

**输入**
```
::: tip
这是一个提示
:::

::: warning
这是一个警告
:::

::: danger
这是一个危险警告
:::

::: details 查看详情
这是一个详情块，在 IE / Edge 中不生效
```
